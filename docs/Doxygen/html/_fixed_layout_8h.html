Detailed Description

The Layout class defines the layout of vertices in neural networks.

/

The FixedLayout class maintains vertices locations (x, y coordinates), distance of every couple vertices, vertices type map (distribution of excitatory and inhibitory neurons), and starter vertices map (distribution of endogenously active neurons).

The FixedLayout class reads all layout information from parameter description file.
